## Dependencies
Debian / Ubuntu
```
sudo apt-get install make gdb
````
Arch
```
sudo pacman -S make gdb
````

## Installation
  
```sh
git clone https://github.com/hanna1488/hnhx.git
cd hnhx
sudo make
```

## Usage
  
```sh
sudo hnhx [process name] [path to the .so file]
```

You can use `ps -A` to list all process names
